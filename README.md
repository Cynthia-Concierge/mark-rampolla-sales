# Mark Rampolla Sales Page

A simple, effective sales page for Mark Rampolla's Freedom First coaching business.

## Live Site
- GitHub: https://github.com/Cynthia-Concierge/mark-rampolla-sales
- Vercel: (pending deployment)

## What's Included

- Clean, modern design focused on conversion
- Mobile responsive
- Fast loading (pure HTML/CSS)
- Clear value proposition and story arc
- Multiple CTAs

## Structure

1. **Hero Section** - Strong headline, clear value prop, immediate CTA
2. **Story Section** - The $200M exit story and transformation
3. **Results Section** - Social proof through numbers
4. **Framework Section** - The 4 pillars of Freedom First
5. **Final CTA** - Email contact for coaching inquiries

## To Deploy

The site is already on GitHub. To deploy to Vercel:

1. Go to https://vercel.com/new
2. Import from GitHub: `Cynthia-Concierge/mark-rampolla-sales`
3. Click "Deploy"
4. Your site will be live at `mark-rampolla-sales.vercel.app`

Optional: Add a custom domain in Vercel settings.

## To Edit

All content is in `index.html`. The page uses:
- Clean typography (Inter font)
- Simple color scheme (blue primary, neutral grays)
- Inline CSS for fast loading

Make edits, commit, and push - Vercel will auto-redeploy.
